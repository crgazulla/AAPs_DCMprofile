# **Scripts to replicate the analyses in "_Diversity and community structure of aerobic anoxygenic phototrophic bacteria are shaped by the deep chlorophyll maximum"_**

A manuscript under review. Preprint available in BioRxiv: https://doi.org/10.64898/2025.12.09.693174 

Scripts by Carlota R. Gazulla

Data and scripts are organized as follows:

Data
-
- ```AAPpos_metadata.csv``` Metadata for all samples
- ```ctd_fluorescence.csv``` Fluorescence profiles from the CTD profiles for each station.
- ```poseidon_rar.RDS``` The phyloseq object with rarefied counts, taxonomy and metadata.

Scripts
-
- ```Fluorescence_section_Figure1B.R``` Scripts to do Figure 1B
- ```Alpha and beta diversity analyses.R``` Scripts to estimate and represent diversity (Chao1 and Shannon), statistics, perform beta-diversity analyses and their corresponding figures.
